# 🔨 Build Mode Protocol

> Khi vào Build Mode, bạn là một cỗ máy. Không social media. Không multitask. Chỉ có code.

## Trước khi bắt đầu

```
1. Đóng tất cả tab không liên quan
2. Phone để im lặng hoặc úp mặt xuống
3. Mở duy nhất:
   - VS Code (project đang build)
   - Browser (localhost + docs nếu cần)
   - Antigravity (AI assistant)
   - Terminal
4. Mở today.md → biết đang làm task gì
5. Set timer: 90 phút hoặc 120 phút
```

## Trong khi build

```
✅ Code → Test → Commit (micro-cycles)
✅ Mỗi 30 phút commit 1 lần (dù nhỏ)
✅ Stuck > 20 phút? → Gọi AI → Nếu vẫn stuck, ghi BLOCKED và switch task
✅ Phát hiện bug không liên quan? → Ghi vào BACKLOG, không fix ngay
✅ Có idea mới? → Ghi vào IDEAS/, quay lại task hiện tại

❌ KHÔNG mở tab mới "chỉ xem nhanh"
❌ KHÔNG refactor code đang chạy tốt
❌ KHÔNG thêm feature vào task đang làm (scope creep)
❌ KHÔNG check tin nhắn giữa session
```

## Sau khi xong

```
1. git add . && git commit && git push
2. Move task file sang đúng status
3. Nghỉ 10-15 phút thật sự
4. Bắt đầu session tiếp hoặc evening shutdown
```

## Nhạc nền khuyến nghị
- lo-fi beats
- Ambient/Focus playlists
- Silence (tốt nhất)

## Metrics theo dõi
- Commits/ngày
- Tasks DONE/tuần
- Ship frequency

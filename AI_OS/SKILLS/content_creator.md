# ✍️ Skill: Content Creator

## Công dụng
Agent giúp viết blog posts, project case studies, social media content, documentation và mọi dạng technical writing.

## Khi nào gọi
- Viết blog post kỹ thuật cho portfolio
- Viết case study cho dự án đã build
- Viết LinkedIn / X posts
- Viết documentation cho code
- Viết README cho repo

## Prompt mẫu

### Technical Blog Post
```
Bạn là Technical Writer + Developer Advocate.

Chủ đề: [topic]
Target audience: Developers 1-3 năm kinh nghiệm
Tone: Conversational, thực tế, có ví dụ code

Viết blog post gồm:
1. Hook mở đầu (tại sao topic này quan trọng)
2. Problem statement
3. Solution approach (có code snippets)
4. Lessons learned
5. Conclusion + CTA

Độ dài: ~800-1200 từ
Format: Markdown
```

### Project Case Study
```
Bạn là Product Writer.

Dự án: [tên]
Stack: [tech]
Vấn đề giải quyết: [problem]
Kết quả: [metrics]

Viết case study format:
1. The Challenge (1 paragraph)
2. The Approach (technical decisions)
3. The Solution (screenshots/architecture)
4. The Impact (numbers)
5. What I'd Do Differently

Tone: Tự tin, data-driven, honest.
```

# 🎯 Skill: Recruiter Optimizer

## Công dụng
Agent giúp tối ưu mọi thứ liên quan đến xin việc: portfolio copy, CV, GitHub profile, LinkedIn, cover letter, và interview prep.

## Khi nào gọi
- Viết mô tả dự án cho portfolio
- Optimize GitHub README profile
- Chuẩn bị cho phỏng vấn
- Review CV/resume
- Viết cover letter
- Tối ưu LinkedIn headline/summary

## Prompt mẫu

### Project Description (cho Portfolio)
```
Bạn là Technical Recruiter + Copywriter.

Tôi có dự án:
- Tên: [tên]
- Stack: [tech]
- Chức năng: [mô tả thô]
- Số liệu: [users, metrics nếu có]

Viết mô tả dự án cho portfolio website với:
1. Headline hấp dẫn (1 dòng, dưới 10 từ)
2. Summary (2-3 câu, focus vào IMPACT không phải feature)
3. Technical challenges solved
4. Key metrics / results

Tone: Tự tin nhưng không khoe khoang. Chuyên nghiệp.
Ngôn ngữ: Tiếng Anh.
```

### CV Bullet Points
```
Bạn là Career Coach chuyên về tech.

Tôi ứng tuyển vị trí: [role]
Kinh nghiệm: [mô tả]

Viết 4-5 bullet points theo format:
"[Action verb] + [What you did] + [Impact/Result]"

Ví dụ: "Architected a real-time data pipeline processing 50K events/sec, reducing latency by 40%."
```

### Interview Prep
```
Bạn là Senior Tech Interviewer tại [company].

Tôi đang ứng tuyển vị trí: [role]
Stack chính: [tech]

Hỏi tôi 5 câu technical interview phổ biến nhất cho role này.
Sau mỗi câu trả lời của tôi, đánh giá và cho feedback.
```

# 💡 Product Feature Ideas

Nơi lưu trữ các tính năng sẽ/có thể xây dựng trong tương lai, được lọc từ phản hồi của User/Nhà tuyển dụng.

## 📝 Idea Format
- **Tên Feature:** [Tên]
- **Value:** [Mang lại giá trị gì cho portfolio?]
- **Complexity:** [Low/Medium/High]
- **Status:** [Draft / Approved / Rejected]

---

### [Draft] Live GitHub Activity Heatmap
- **Value:** Cho thấy sự chăm chỉ code mỗi ngày ngay trên Homepage.
- **Complexity:** Medium (Cần fetch GitHub API và làm cache).

### [Draft] Admin Analytics Dashboard
- **Value:** Biết chính xác ai vào, click vào đâu.
- **Complexity:** High (Cần set up Vercel Analytics / PostHog).
